# QPlugin <img width=50 src="/src/icon.png"/> 
Plugin para executar ações do android por Python3

### Como usar:

* Instale o QPlugin.apk ou importe o projeto do Tasker
* Abra a aplicação que ela mostrará uma notificação de serviço
